# Pika-Pika

## 기술 스택
- .Net 8.0
- WPF
- CommunityToolkit
- NUnit

## V1
- 1대 1 UDP 통신 피카츄 배구
- 하나의 게임을 만들어본다.

### 기능
- [ ] 대전 상대가 없으면 대기해야 함. (한무로딩)
- [ ] 대전 상대가 나타나면 3초후 게임 시작. (도망불가)
- [ ] UDP 통신을 사용한 배구 대전.
